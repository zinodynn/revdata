# Changelog

## 2026-04-28

- Fixed the issue of directory filtering: When clicking on the directory, it now supports recursive loading of data sets in subdirectories.
- Added the ability to select multiple for export and batch deletion, allowing for the export of data sets after approval.
- Added the "Export" operation in the right-click menu of the directory, enabling batch export at the directory level.
- Optimized the logic for deleting data sets: If there are unfinished tasks, before deletion, confirmation and task assignment can be canceled.
- Added a "Task Management" tab for viewing and managing the delegated task relationships within the data set.
